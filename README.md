# How to Use

## step 1 
Add the JitPack repository to your build file

```
	allprojects {
			repositories {
				...
				maven { url 'https://jitpack.io' }
			}
		}
```

## Step 2
Add the dependency

```
dependencies {
	        compile 'com.github.itlwy:DBindingView:v1.0.1'
	}

```
